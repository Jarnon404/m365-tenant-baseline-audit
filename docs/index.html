M365 Tenant Baseline Audit

Public-safe PowerShell toolkit for read-only Microsoft 365 tenant baseline review. Designed for Global Reader style audits, sanitized reporting and practical IT operations.

Read-only enforced

The script validates the active Microsoft Graph context and stops if write-capable scopes are present. It collects baseline information and creates local reports only.

Global Reader style access

Sections complete where delegated read permissions are available and are marked as skipped where tenant consent or service access is not available.

Reporting

Outputs HTML, CSV, TXT and JSON reports with sanitized mode enabled by default.

Audit areas

  • Tenant basics and verified domains
  • Licensing and organization sync visibility
  • Users, guests, groups, devices and directory roles
  • Conditional Access and authentication methods policy visibility
  • Application registrations and enterprise applications
  • Intune managed devices, policies and app inventory
  • Optional Exchange Online read-only baseline

Read-only guard

The recommended model is a dedicated read-only Entra ID app registration. If the Microsoft Graph context contains ReadWrite, Write, FullControl or similar broad scopes, the audit stops before collection starts.

Usage

.\scripts\Invoke-M365TenantBaselineAudit.ps1 -TenantId "<TENANT_ID>" -ClientId "<READ_ONLY_APP_CLIENT_ID>" -OutputPath C:\Temp\M365TenantBaselineAudit
